# Petals Backend

This is the backend for the petals app, made with nodeJs and mongodb.

## To run the project

1. Start Mongodb

    ```
    sudo systemctl start mongodb
    mongo
    ```

2. Start server

    ```
    cd backend
    npm i
    npx nodemon index
    ```
